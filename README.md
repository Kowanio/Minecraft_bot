# Minecraft Bot
MinecraftBot is a little bot write in Python

## Configuration


You can choose `activated` if you want feed RSS

```json
{
  "discord_bot": {
      "token": "",
      "prefix": "",
      "description": ""
  },
  "feed": {
      "feed_channel": 0,
      "feed_url": "https://www.minecraft.net/en-us/feeds/community-content/rss",
      "activated": true
  }
}
```
